Confetti.js

About the library

This library enables you to throw confetti on yout website, and of course, it is open sourced and free to everyone who wants it! Cool, isn't it?

It is made using Vanilla JS, so you are not required to acquire any additional plug ins, libraries or frameworks. It requires for a canvas HTML element to be specified as it uses canvas context to draw confetti and make a party. This was initially made for my portfolio website to fire up on my birthday!

Getting Started

Initializing the party

You have control over most of the things here. You can set options and configure your confetti party. Options are sent to the party when the party is initialized, and we will look on how to do that later on. Now, we just wanna declare some options. We will dive in explanation for each particular option later on, stay tuned.

const options = {
id: "confetti-canvas",
number: 200,
speed: 500,
wind: 250,
colors: ["red", "blue"],
shapes: ["triangle"],
fullscreen: true,
removeAfterEnd: false
zIndex: 100,
display: "block",
};

Now that we have declared JSON variable that will represent our options, only thing left is to initalize our party:

const confetti_party = new ConfettiParty(options);

Now that our party is declared, we can start it using start() method:

confetti_party.start();

Options in depth

id

This options is used to specify the ID attribute of canvas element you want your party to use.
<canvas id="confetti_party"></canvas>

number

Specifies how much confetties to generate.
Default: 500

speed

Specifies how fast should confetties fly. If the value is, let's say, 2500, confetties will fly by speed in range of 1250 to 3750.
The formula is speed / 2 to speed * 3 / 2
Default: 2500

wind

Specifies how fast should confetties be carried left or right by the wind. If the value is, let's say, 250, confetties will be carried by wind in range of 125 to 375. Script randomly chooses if confetti will fly left or right.
The formula is wind / 2 to wind * 3 / 2
Default: 250

colors

An array representing colors for confetties. Each confetti is assigned a random color from the colors array.
Default: ["red", "lightgreen", "lightblue", "yellow"]

shapes

An array representing shapes for confetties. Each confetti is assigned a random shape from the shapes array. All CSS color values are available.
Available: rectangle, triangle, circle
Default: ["rectangle", "triangle", "circle"]

fullscreen

Boolean value indicating if the canvas should be full screen or not. The script will take window width and height and set for canvas position to be fixed and for canvas parent element to be relatively positioned so the fullscreen effect could be achieved.
Available: true, false
Default: false

removeAfterEnd

Boolean value indicating if the canvas element should be hidden and sent to back when animation finishes. If this is true, display will be none and z-index property will be -1.
Available: true, false
Default: false

zIndex

A number indicating which z-index property should the canvas element get when the party starts.
Default: 1

display

String value indicating how to display the canvas when party starts.
Available: CSS values for display
Default: block
